## Getting Started

### Play Locally
1. Clone this repository
2. Open `index.html` in a modern web browser
3. No build process needed - it's pure HTML/CSS/JS!

### Deployment
This project is configured for Netlify deployment:
- Push to GitHub
- Connect your repository to Netlify
- It will auto-deploy from the root directory

## Live Demo
[Play Now](https://brutalspan.netlify.app)

## How to Play
1. Choose your judgment intensity (Gentle/Honest/Brutal)
2. Select language (English/Hinglish)
3. Memorize the numbers shown
4. Click them in ascending order when they disappear
5. Survive as long as you can!
