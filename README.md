# Recall - Memory Roast Game

A neon-themed memory game that roasts you when you fail! Dynamic roasts in English and Hinglish with meme displays.

## Features
- 380+ dynamic roasts from Roast.md
- 50+ meme GIFs
- Language support: English & Hinglish
- Progressive difficulty
- Neon cyberpunk aesthetic
